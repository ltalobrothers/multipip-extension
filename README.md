# MultiPIP - 多视频画中画浏览器插件

一个支持同时显示多个网页视频的画中画功能的Edge浏览器插件。

## 🚀 功能特性

- **多视频支持**：同时创建多个画中画窗口
- **自动检测**：自动扫描网页中的视频元素
- **跨网页工作**：支持不同网站的视频
- **窗口管理**：可拖拽、调整大小、置顶显示
- **音频控制**：独立控制每个视频的音量
- **智能布局**：自动排列多个画中画窗口

## 🎯 核心功能

### 1. 多视频画中画

- 同时显示多个视频窗口
- 每个窗口独立控制
- 支持拖拽、调整大小

### 2. HLS流支持

- 支持 `.m3u8` 直播流
- 自动加载 Hls.js 库
- 直播/点播都支持

### 3. 智能视频检测

- 深度扫描页面视频
- 支持 Canvas、WebGL 视频
- 自动识别视频类型

### 4. 高级控制

- 音量独立控制
- 播放速度调整
- 画质切换（HLS）
- 全屏模式

## 📦 安装方法

### 1. 准备图标文件（也可以不准备）

在项目根目录下创建 `icons` 文件夹，并添加以下图标文件：

- `icon16.png` - 16x16 像素图标
- `icon48.png` - 48x48 像素图标
- `icon128.png` - 128x128 像素图标

你可以使用任何图片编辑器创建这些图标，或者使用在线图标生成工具。

### 2. 加载插件到Edge浏览器

1. 打开Edge浏览器，在地址栏输入 `edge://extensions`
2. 在页面右上角开启「开发人员模式」
3. 点击「加载解压缩的扩展」
4. 选择本项目的文件夹
5. 插件安装成功！

## 🎮 使用方法

1. 打开包含视频的网页
2. 点击浏览器工具栏中的 MultiPIP 插件图标
3. 点击「扫描当前页面视频」按钮
4. 在扫描结果中，点击视频旁边的「画中画」按钮
5. 视频将以画中画模式显示在屏幕上
6. 你可以重复步骤4，打开多个画中画窗口

## 🔧 技术栈

- Manifest V3
- JavaScript (ES6+)
- Hls.js (用于HLS流支持)
- Picture-in-Picture API

## ⚠️ 注意事项

1. 某些网站可能有自定义的视频播放器，可能需要先点击播放按钮让视频加载后再扫描
2. 浏览器原生的画中画API通常只支持同时显示一个画中画窗口，如需同时显示多个，插件会使用自定义的画中画容器
3. HLS流支持需要网络连接来加载Hls.js库
